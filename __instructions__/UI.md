Goal
Build a UI using Angular JS for the multi-agent system.

1. The UI is a responsive web application that can be accessed from any device.
2. This UI is for a user who is looking to buy a car.
3. The user can search for cars based on their needs and preferences.
4. The UI will only integrate with the ADK agent.
5. The UI will have a chat interface where the user can chat with the agent.
6. Create the UI ciode under a folder called web.
